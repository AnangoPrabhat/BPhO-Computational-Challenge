This tool simulates how an eye focuses an image and how glasses can correct vision. A positive (+) inherent error represents Myopia (short-sightedness), and a negative (-) inherent error represents Hyperopia (long-sightedness).

Eye & Object Configuration:

E.g., +2.0 for 2D Myopia, -1.5 for 1.5D Hyperopia.
2 m

Correction Mode:

Shows the eye's vision without any correction.

Simulates wearing glasses. Set the power of the corrective lens. For a perfect correction, the glasses power should be the negative of the eye's inherent error (e.g., +2.0D error needs -2.0D glasses).

Ray Diagram Legend

Purple Rays: Show the path of light through the uncorrected eye.

Orange Rays: Show how the glasses bend light *before* it reaches the eye.

When glasses are active, the purple rays show how the eye focuses the light that has *already been bent* by the glasses.
